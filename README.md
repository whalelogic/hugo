## üöÄ Running Locally

To run this site on your local machine, you'll need to have [Hugo installed](https://gohugo.io/getting-started/installing/).

1.  **Clone the repository:**
    ```sh
    git clone [https://github.com/whalelogic/hugo.git](https://github.com/whalelogic/hugo.git)
    cd hugo
    ```

2.  **Install Theme Modules:**
    This theme uses Hugo's module system. You may need to initialize it the first time.
    ```sh
    hugo mod tidy
    ```

3.  **Start the Hugo Server:**
    Run the built-in Hugo server. It will automatically watch for file changes and rebuild the site.
    ```sh
    hugo server
    ```

4.  **View Your Site:**
    Open your browser and go to `http://localhost:1313`.
